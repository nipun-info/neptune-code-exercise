Differnt Units

  1. % : Relative to the value of parent element. 100% is the width of the parent element
  2. px : Pretty self explanatory. Absolute length in pixel.
  3. em : Relative to the font-size of the parent element.
  4. rem : Relative to font-size of the root element.
  5. vh : equal to 1% of the height of the browser window size.
  6. vw : equal to 1% of the width of the browser window size.
